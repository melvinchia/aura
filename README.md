# AURA - Age-aware Understanding & Resourceful Assessment

**A gentle, private, and respectful initial screening tool for ADHD and Autism traits.**

## About AURA

AURA is a free, static, client-side web application designed to help individuals and families explore possible ADHD or Autism traits in a friendly, age-appropriate way.

### Key Features
- Four age groups: 7–12, 13–20, 21–40, and 41+
- Separate question pools for ADHD and Autism (200 questions per condition)
- 15 randomized questions per condition (30 total)
- Support for "Yes", "No", and "Sometimes" answers
- Smart replacement for "Sometimes" responses
- Keyboard navigation support
- Personalized results with warning when too many "Sometimes" are selected

**Important**: AURA is **not a diagnostic tool**. It is an informal screening aid only. Professional assessment by a qualified clinician is strongly recommended.

## How to Use

1. Go to the [live site](https://melvinchia.github.io/aura/)
2. Enter your name and select your age group
3. Answer the questions (mouse or keyboard)
4. View your trait scores (1–10) for ADHD and Autism

## Self-Hosting / Development

1. Clone or download this repository
2. Make sure `questions.json` is in the root folder
3. Open `index.html` in a browser or host it on GitHub Pages
